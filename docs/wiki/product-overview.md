# Product Overview

## 제품 성격

이 프로젝트는 일본어 단어 학습 앱과 단어장 편집기를 함께 가진다.

- 메인 앱 진입점: `/`
- 에디터 진입점: `/editor`

메인 앱은 학습용이고, 에디터는 데이터 관리용이다. 둘은 같은 데이터 원천을 공유하지만 사용 방식은 다르다.

## 핵심 사용자 흐름

### 메인 앱

1. 홈에서 목록, 학습, 공유 중 하나를 고른다.
2. 학습 메뉴에서 일반 학습, 동사 활용, 시험, 게임으로 이동한다.
3. 세션이 남아 있으면 홈 배너에서 이어갈 수 있다.

### 에디터

1. 기본, 주제형, 비교형 중 하나의 데이터 모드를 고른다.
2. 좌측에서 세트를 선택한다.
3. 중앙 표에서 직접 수정한다.
4. 로컬 파일 또는 워크스페이스로 내보낸다.

## 기능 축

- 목록: 단어를 빠르게 훑고 가리기/검색/즐겨찾기 중심으로 본다.
- 일반 학습: 셔플된 항목을 반복 라운드로 학습한다.
- 동사 활용: 활용형 문제를 풀고 오답만 다시 학습할 수 있다.
- 시험: 자동 채점과 직접 채점을 지원한다.
- 게임: 안정 범위에서는 스피드 퀴즈만 문서화한다.
- 공유: 앱 백업을 클립보드, 파일, QR로 다룬다.

## 사용자 경험 원칙

- 텍스트 설명은 최소화한다.
- 아이콘만 봐도 기능이 추론되도록 만든다.
- 선택 상태와 주요 액션은 가능한 한 현재 표면에서 드러난다.
- 별도 상세 패널 의존을 줄인다.
