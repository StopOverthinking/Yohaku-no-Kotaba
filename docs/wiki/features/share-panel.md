# Share Panel

## 목적

앱 전체 백업을 한 화면에서 제공한다.

## 진입점

- 홈 서브메뉴
- 컴포넌트: `src/features/share/SharePanel.tsx`

## 앱 섹션 액션

- 클립보드 복사
- 파일 저장
- QR 내보내기
- 클립보드 복원
- 파일 복원
- QR 복원

## QR 동작

- 큰 payload는 프레임 분할
- SVG QR 생성
- 여러 장이면 자동 재생
- 카메라 스캔은 브라우저 `BarcodeDetector`와 `getUserMedia` 사용

## 복원 흐름

- JSON 파싱
- 확인 모달
- 기존 `jsp-react:` 데이터 교체
- 페이지 새로고침

## UI 규칙

- 레이블은 짧게
- 버튼 의미는 아이콘 중심으로 바로 보이게
- 긴 설명은 확인 모달에서만 사용

## 수정 시 주의점

- 제거된 기능의 레거시 키가 앱 백업에 다시 섞이지 않게 한다.
- QR 복원은 앱 백업 포맷만 처리한다.
