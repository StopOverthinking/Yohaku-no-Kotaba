# Current State

작성 기준: 2026-04-24

## 최근 반영 완료

- 시험 결과가 0오답이어도 기존 오답 노트 ID를 빈 배열로 덮어쓰지 않음
- 예전 시험 결과의 오답 항목 형태를 로드 시 정규화하고 오답 노트 ID를 복구
- 시험 설정의 최근 결과 패널을 오답 노트와 분리해서 삭제 가능
- 홈의 최근 시험 결과 배너도 오답 노트와 분리해서 삭제 가능
- 일반 학습 `이전 카드`가 연속 undo 가능
- 학습 카드 전환 시 다음 카드가 너무 빨리 보이던 움찔 현상 완화
- 터치 후 툴팁 포커스 잔류 수정
- 앱 전반의 고비용 시각 효과 축소
- 목록 검색, 가리기, 글자 크기 변경 반응성 개선
- 목록의 일본어/뜻 가리기를 실제 마스킹 바로 변경
- 일반 학습 완료 화면 액션을 홈 이동 1개로 단순화
- 스마트 복습이 `IndexedDB` 일정 저장 구조로 이동
- 비교형 단어장을 학습/시험에서 제거하고 게임·스마트 복습 후보에서도 제외
- 단어장 에디터에서 기본/주제형 스마트 복습 예문을 표 전환과 xlsx로 조회/편집 가능
- 스마트 복습 예문 자동 생성과 어조 메타데이터 제거, 직접 작성 예문만 세션 후보로 사용

## 현재 우선순위

- 목록 모드에서 `전체 단어장` 제거
- 스마트 복습에서 학습할 단어 개수 지정 UI 연동
- 진행 중인 학습/시험 파기 팝업을 브라우저 팝업 대신 앱 내부 팝업으로 교체
- 학습 설정 저장 누락 여부 확인
- 공유 메뉴의 불필요한 텍스트와 공간 낭비 축소

## 계속 지켜야 할 주의점

- 미니멀, 아이콘 중심 UI 유지
- 설명 문구 늘리기보다 상태 표현 개선 우선
- blur, filter, height animation, backdrop-filter 재확대 금지에 가깝게 신중히
- 에디터는 모바일 대응보다 데스크톱 표 편집성 우선
- `탭 매치 러시`는 미완이므로 기능 판단 근거에서 제외
